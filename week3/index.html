Week 3: Hypothesis Testing & Correlation

Statistical inference — from null hypotheses to p-values, and testing relationships between variables

What is Hypothesis Testing?

Hypothesis — a proposed explanation for a phenomenon.
Hypothesis testing — a statistical procedure to test whether a hypothesis can be rejected or not.

Example question: Is there an association between recovery time after surgery and the sex of the dog?

Null Hypothesis ($H_0$) vs Alternative Hypothesis ($H_a$)

$H_0$ — Null Hypothesis

States that there is no relationship / association / difference between the variables under investigation.

Example: There is no difference in average recovery time between male and female dogs.

$H_a$ — Alternative Hypothesis

States that there is some kind of relation or association.

Example: There is a difference in average recovery time between male and female dogs. Male dogs have longer recovery time.

Type I & Type II Errors

$H_0$ is TRUE $H_0$ is FALSE
Reject $H_0$ Type I Error (False Positive)
$\alpha$ = significance level		 Correct decision!
Power = $1 - \beta$
Not reject $H_0$ Correct decision Type II Error (False Negative)
$\beta$ = type II error rate
Type I Error ($\alpha$): Asserting a difference when there is none (false alarm). Typically set at $\alpha = 0.05$ (5%).
Type II Error ($\beta$): Failing to detect a real difference. Related to statistical power ($1-\beta$) — the probability of detecting a true effect. Larger sample size → higher power.

P-Value

The p-value is the probability of observing the data (or more extreme) assuming the null hypothesis is true.

  • Smaller p-value → stronger evidence against $H_0$ (higher significance)
  • When $H_0$ is true, p-value is the cumulative probability of finding this or more extreme events at the tail end
  • Reject $H_0$ if $p < \alpha$ (we observed an extreme event)
Interpretation:
If $p < \alpha$ (e.g., $p = 0.03 < 0.05$): "There is a significant association/difference" (reject $H_0$)
If $p \ge \alpha$ (e.g., $p = 0.30 > 0.05$): "No significant association can be identified" (do not reject $H_0$)

One-Tailed vs Two-Tailed Tests

Two-Tailed Test

A test for whether there is a difference between groups (direction unspecified).

"There is a difference in recovery time between male and female dogs."

One-Tailed Test

A test for whether there is a difference in a specific direction.

"Male dogs have lower recovery time than female dogs."

Five Steps in Hypothesis Testing

  1. State the hypotheses: Define $H_0$ and $H_a$
  2. Choose a test method and set $\alpha$: Different data types need different tests; $\alpha$ is usually 0.05, 0.01, or 0.001
  3. Compute the test statistic: Summarise evidence according to the chosen test
  4. Calculate p-value: And compare it to $\alpha$
  5. Make a decision: Reject $H_0$ or not

Example: The Coin Flip

We toss a fair coin 8 times and observe 7 heads, 1 tail. Is the coin fair?

Step 1: $H_0$: The coin is fair. $H_a$: The coin is loaded towards heads.
Step 2: Binomial distribution with $p=0.5$, $n=8$; $\alpha=0.05$, one-tailed test.
Step 3-4: $P(|X \ge 7|) = P(7) + P(8) = 0.0312 + 0.0039 = 0.0351$
Step 5: $p = 0.0351 < \alpha = 0.05$ → Reject $H_0$

The Monte Carlo & Gambler's Fallacy

In 1913, at Monte Carlo Casino, the roulette ball fell on black 26 times in a row. Gamblers bet everything on red, believing it was "due."

The fallacy: Past independent events don't affect future probabilities. Each spin is independent — the ball doesn't "remember."

Lesson: "Failing to reject" does not mean "proving" the null hypothesis. Just because we haven't found evidence against $H_0$ doesn't mean $H_0$ is true. A small sample may lack the power to detect a real effect.

Correlation: Testing Association Between Two Continuous Variables

Pearson Correlation Coefficient ($r$)

A measure of linear correlation between two continuous variables X and Y, ranging from -1 to +1.

$$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}$$

Spearman's Correlation ($\rho$)

Also called Spearman's rank correlation. Non-parametric — uses ranks instead of raw values. More robust to outliers and non-linear relationships.

Use Pearson if:

  • Data is normally distributed
  • Linear relationship
  • Few outliers

Use Spearman if:

  • Data is skewed or has outliers
  • Non-linear monotonic relationship
  • Ordinal data involved

Correlation ≠ Causation

Ice cream sales and drowning deaths are strongly correlated — but ice cream doesn't cause drowning. The hidden variable is summer weather (hot days → more ice cream, more swimming → more drownings).

Always remember: A significant correlation ($p < 0.05$) means there is a statistical association — it does NOT prove causation. Causation requires experimental design, control groups, and randomisation (covered in Week 12).

How to Report Results

  • Always report the actual p-value: say "$p = 0.03$", not "$p < 0.05$"
  • State the test used: "... (p = 0.002, t-test)" or "... (Spearman's $\rho = -0.89$, p < 0.001)"
  • Include descriptive stats: "Male (mean: 1.72; SD: 0.03) and female (mean: 1.62; SD: 0.02) dogs"
  • Don't talk about "accepting" the null: just report findings — "no significant difference was found"
